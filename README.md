# KaboomTyper
KaboomTyper is currently a Windows only application. Type words fast and correctly to blow them up. Fail too often and they stackup and you loose!

# Getting Started
- git clone https://github.com/FMock/KaboomTyper.git
- Run PreBuild.bat located in the scripts folder, KaboomTyper\Scripts
- Build the solution using Visual Studio 2022 as Release or Debug x64
- Set either either the LegacyKaboomTyper or the KaboomTyper project as the startup project
- CTRL-F5 to run application
